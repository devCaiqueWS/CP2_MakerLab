# Manual de montagem — Caixa mecânica com engrenagens

Projeto da atividade prática do **Project-based Maker Lab** (Profª Gedeane).

Contato: **profgedeane.kenshima@fiap.com.br**

## Objetivo

Montar e medir as peças para entender como funciona o conjunto mecânico da caixa com engrenagens.

## Peças recebidas

- Caixa inferior (2)
- Tampa (1)
- Engrenagens (10)
- Eixos e encaixes (2)
- Componentes móveis (5)

## O que cada grupo deve entregar

- Repositório no GitHub ensinando a montagem da caixa para um usuário leigo
- Medições das principais peças com paquímetro, registradas neste repositório
- Sistema mecânico montado e testado
- Escolha de uma peça do conjunto para modelagem 3D
- Fotos do passo a passo da montagem

## Materiais necessários

- Kit da caixa mecânica com engrenagens
- Paquímetro
- Celular para registro das fotos
- Bloco de notas para anotações de medidas

## Passo a passo de montagem (para usuário leigo)

1. **Separe e organize as peças**  
   Coloque todas as peças sobre a mesa e confira as quantidades da lista.
2. **Identifique a base da caixa**  
   Escolha uma caixa inferior para começar a montagem.
3. **Posicione eixos e encaixes**  
   Insira os eixos nos pontos de suporte da caixa, sem forçar.
4. **Monte as engrenagens nos eixos**  
   Encaixe as engrenagens uma a uma, verificando se os dentes se tocam corretamente.
5. **Adicione componentes móveis**  
   Coloque os componentes móveis nos locais previstos do mecanismo.
6. **Teste o movimento**  
   Gire manualmente o eixo principal e observe se o conjunto se move sem travar.
7. **Fechamento com a tampa**  
   Posicione a tampa e verifique se há contato indevido com engrenagens/eixos.
8. **Teste final**  
   Gire novamente o sistema já fechado para confirmar o funcionamento.

## Checklist de observação durante a montagem

- [ ] O encaixe está frouxo ou apertado?
- [ ] O eixo gira livremente?
- [ ] As engrenagens engatam corretamente?
- [ ] Existe ponto de contato com a parede da caixa?
- [ ] O movimento trava em algum ponto?
